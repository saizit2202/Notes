<div>   2.  Arrow function:</div>
<div><br></div>
<div>              These are new way to declare the function</div>
<div><br></div>
<div>             Old: </div>
<div><br></div>
<div>              hello = function() {</div>
<div>               return "Hello World!";</div>
<div>              }</div>
<div><br></div>
<div>              New : </div>
<div><br></div>
<div>	      	hello = () => {</div>
<div>		  return "Hello World!";</div>
<div>		}</div>
<div><br></div>
<div>           hello = () => "Hello World!";  ->  If the function has only one statement, and the statement returns a value, you can remove the brackets <i>and</i> the return keyword:</div>
<div><br></div>
<div>	   hello = (val) => "Hello " + val; -> With parameter  </div>
<div><br></div>
<div>	   hello = val => "Hello " + val;  -> with only one parameter, we can remove the bracket</div>
<div><br></div>
<div>        Using “This” inside the arrow function will represent the owner of the function.</div>
